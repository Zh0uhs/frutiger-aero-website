# Music Folder

Place your MP3 files in this folder.

## File Naming Convention

The music player expects these files:
- `dream-land.mp3` - Track 1
- `ocean-waves.mp3` - Track 2
- `forest-walk.mp3` - Track 3

## Supported Formats

- MP3 (recommended)
- WAV
- OGG

## Notes

- File names must match exactly (lowercase with hyphens)
- For Chinese characters in filenames, the player may not display them correctly
- Rename your files to English names for best compatibility
